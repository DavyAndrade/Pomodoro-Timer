# Fluxo

Tela Inicial: Lista de Tarefas, podendo editar a tarefa, deletar a tarefa e adicionar uma nova tarefa. Além disso, clicar na tarefa faz com que ela seja selecionada e entre no pomodoro.

Tela de Configurações: Tela de configurações, podendo editar o tempo do pomodoro, a quantidade de pausas e o tempo de cada pausa.
